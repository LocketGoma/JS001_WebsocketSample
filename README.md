# JS001_WebsocketSample
자바스크립트 웹소켓 연습 1

>>테스트용으로만 사용할것<<

서버 실행법 :

1) bin/www 폴더에 js 파일 링킹 

(링킹 방법 :
const webSocket = require("../wsServer"); 
 webSocket(server); ) 을 WWW 파일에 입력. 


2) js 파일 위치에서 node.js 터미널 실행

3) npm start
